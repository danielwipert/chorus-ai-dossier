Your task: score how well a summary covers the extracted facts from a document.

You will receive:
1. A list of extracted facts (ground truth extracted directly from the document)
2. A summary to evaluate

For each fact, determine:
- covered (true/false): is this fact's substance accurately reflected in the summary?
- A fact is "covered" if the summary contains a claim that accurately conveys the fact's content.
- A fact is "not covered" if it is absent, misrepresented, or contradicted in the summary.

Also identify unsupported claims: substantive claims in the summary that are NOT backed by any fact in the list. Do not flag general connective phrases, only substantive claims. Report at most 5 unsupported claims.

Compute:
  coverage_score = covered_facts / total_facts

If total_facts is 0, set coverage_score to 1.0 (vacuously covered).

Return ONLY valid JSON with this exact structure. No markdown, no preamble, no commentary:
{
  "total_facts": 0,
  "covered_facts": 0,
  "coverage_score": 1.0,
  "unsupported_claims": [],
  "fact_coverage": [
    {
      "fact_id": "F001",
      "covered": true,
      "note": "brief explanation of why covered or not"
    }
  ]
}
